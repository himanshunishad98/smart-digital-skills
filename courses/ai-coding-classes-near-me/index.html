FAQ

Frequently Asked Questions

Everything you need to know about our classes, schedules, and certificate criteria.

Who is eligible to join the coding and AI classes? +
Any middle-school student currently in Class 6, 7, or 8 (ages 10 to 14) is welcome. No prior coding experience is required; we start from the very basics of algorithmic thinking.
What programming languages or tools will my child learn? +
Students learn programming fundamentals using Scratch (block coding) to build interactive games. This provides a visual foundation for variables, loops, events, and conditional logic before advancing to text-based languages like Python.
What coding projects will my child build? +
Kids build real programming portfolios, including a coordinate-based maze game, an arcade game with score tracking, and simple AI-assisted prompting scripts.
How do coding classes help my child academically? +
Coding logic directly aligns with math and science concepts like coordinate geometry, variables, and sorting algorithms. This supports the CBSE computer applications syllabus and suggestions in the NEP 2020 framework.
